## サービス概要
- CookLogは、「平日に2〜3回は自炊したい」と思っている人のための料理習慣化サポートアプリです。  
- 目標となる1週間の料理回数を設定し、作った料理を写真・料理名・メモ付きで記録できます。  
- 記録はダッシュボードで可視化され、目標達成度が一目でわかり、無理なく料理を続けることをサポートします

## 開発背景
料理を習慣化できると日々の食事がもっと楽しくなる、
そう気づいた自分自身の経験がこのアプリを開発するきっかけです。

もともと料理はまったくしたことがなく、レタスとキャベツの違いが本気でわからない、
イチョウ切りはイチョウの茎の部分まで再現する切り方だと思っていた、
そもそも味音痴で何でもおいしく感じる、など散々でした。

そんな自分でも、平日の夜ご飯を作る機会が増えてからは、
SNSにあるたくさんのレシピを使用して毎日いろいろな料理にチャレンジしたり、
食材の目利きができるようになったり、
味の違いから調味料の工夫がわかるようになったり、「食」に対しての発見が一気に増えました。

「食」を語るには経験も知識もまだまだ足りませんが、
料理をしたい気持ちはあるけれどなかなか続かない人をサポートすることなら、自分にもできるのではないか。
そう考え、このCookLogというアプリを作ろうと思い至りました。

料理だけに限りませんが、アプリ1つで料理を習慣化することは難しいと思います。
それでも、料理特化の習慣化をサポートする機能や仕組みを通じて
料理を習慣にできるきっかけを増やしたいと思い、開発を行っています。

## ユーザー層について
### 想定ユーザー
- 社会人で一人暮らし、または同棲・夫婦暮らし
- 外食・コンビニは多いが、「平日2〜3日くらいは料理をしたい」と思っている
- 仕事に疲れてなかなか料理が継続できていない人

### なぜこの層を対象にしたか
- 自分自身が、料理を続けたくてもなかなか続けられなかった経験があるため
- 私と同じように「料理を続けたくても続けられない」人が多くいることがわかったため

## ユーザーの獲得について
- Xの活用、料理をする方に届くような投稿を行う
- 自らアプリのユーザーとして日々記録をし、その様子をシェアする
- 友人や家族に共有し、実際に使ってもらう

## サービスの差別化ポイント・推しポイント
- 「週間の料理目標回数」にフォーカス
	- 毎日料理をせずとも「平日に2〜3回できればOK」という現実的な習慣化を想定
	- 自分で決めた目標回数に対する達成状況がダッシュボードで一目で分かる
- 料理特化の習慣記録
  - 汎用的な習慣アプリだと○/×のチェックになってしまうが、
	  CookLogは「料理写真＋料理名」で記録を残せるため、いつ・何を作ったかをセットで振り返れる
- シンプルな機能・UI
  - 習慣化にフォーカスし、高度なレシピ機能や献立作成機能は行わない
