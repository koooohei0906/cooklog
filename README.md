## サービス概要
CookLogは「料理をしたい」と思ったときの“始めるまでのハードル”を下げることにフォーカスしたアプリケーションです。

主な機能は以下の3つです。
- 献立の検討/決定支援
- 買い物リスト
- 料理記録

## 開発背景
料理をする機会が増えると日々の食事がもっと楽しくなる、
そう気づいた自分自身の経験がこのアプリを開発するきっかけです。

もともと私自身、料理をしたことが全くなく、
レタスとキャベツの違いが本気でわからないし、
イチョウ切りはイチョウの茎の部分まで再現する切り方だと思っていたし、
そもそも味音痴で何でもおいしく感じる、など散々でした。

そんな自分でも、ご飯を作る機会が増えてからは、
SNSにあるたくさんのレシピを使用して毎日いろいろな料理にチャレンジしたり、
食材の目利きができるようになったり、
味の違いから調味料の工夫がわかるようになったり、「食」に対しての発見が一気に増えました。

一方で、料理を続ける上で一番しんどいのは「作り始めるまで」「何を作るか決めるまで」だと感じています。  
「食」を語るには経験も知識もまだまだ足りませんが、
料理をしたい気持ちはあるけれどなかなか続かない人をサポートすることなら、自分にもできるのではないか。
そう考え、このCookLogというアプリを作ろうと思い至りました。

## ユーザー層について
### 想定ユーザー
- 社会人（特に優良レシピを見つけられてもなかなか実際に料理をするまでには至れていない方）
- 料理を続けたい気持ちはあるが、献立を考えたり買い物準備が負担になりやすい人

### なぜこの層を対象にしたか
- 自分自身が、料理を続けたくてもなかなか続けられなかった経験があるため
- 私と同じように「料理を続けたくても続けられない」人が多くいることがわかったため

## ユーザーの獲得について
- Xの活用、料理をする方に届くような投稿を行う
- 自らアプリのユーザーとして日々記録をし、その様子をシェアする
- 友人や家族に共有し、実際に使ってもらう

## サービスの差別化ポイント・推しポイント
- 料理をしたい人の「料理を始めるまでのハードルを下げること」にフォーカス
	- 「献立を決める」「買い物をする」に対してそれぞれサポート機能あり
	- 頑張って作った料理もアプリ内で記録・管理出来る
- シンプルな機能・UI
  - 習慣化にフォーカスし、高度なレシピ機能や献立作成機能は行わない

## 機能候補
