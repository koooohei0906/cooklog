## サービス名
CookLog（クックログ）

## サービス概要
料理を続けたいと思っているものの、なかなか続けられない社会人向けに料理習慣化をサポートするアプリです。
料理記録を残し、カレンダー表示や検索で振り返りやすくすることで、料理を習慣化出来ている！と実感できるようになります。

## 開発背景
私自身、たまに料理をする機会はあるものの仕事の帰りが遅くなってしまったり、疲れて外食に頼ってしまうことが多くありました。
外食が悪いわけではありませんが、料理を続けたい中でなかなか続けられない自分が嫌だったこともあり、
料理に特化して料理習慣をサポートしてくれるアプリがあったらいいなと思い、本アプリを開発しようと思い立ちました。
もちろん習慣化はアプリだけで解決するものだとは思っていませんが、
アプリを通じた様々な仕組みで習慣化のサポートをしたいという気持ちで開発をしています。

もともと食べることが大好きでしたが、料理が継続できるようになった今、
食材や味のことにも意識が向き、これまで以上に「食」の時間がより楽しくなっています！

## ユーザー層について
### 想定ユーザー
- 20〜30代の社会人
- 料理をしたい気持ちはあるものの、仕事の疲れ、帰宅途中にある数々の誘惑に打ち勝てずに料理が出来ていない人
- 料理は時々するものの、モチベーションが不安定でなかなか続けられない人

### なぜこの層を対象にしたか
- 自分自身が料理を続けたくてもなかなか続けられなかったため
- 私と同じように料理を続けたくても続けられない人が多くいるため

## ユーザーの獲得について
- Xの活用、料理をする方に届くような投稿をする
- 自らアプリのユーザーとして日々記録をしシェアする
- 友人や家族に共有する

## サービスの差別化ポイント・推しポイント
- 記録が振り返りやすい
  - 汎用的な習慣アプリだと○/×の記録になってしまうが、
　  料理記録は「料理写真＋料理名」で表示されるので、何を作ったかまで振り返りやすい
- シンプルな機能・UI
  - 習慣化をすることにフォーカスし、スムーズな習慣化をサポート
  - 無理に機能を詰め込みすぎず、任せるところは他の既存アプリに任せる

## 機能候補
### MVPリリース時に実装したい機能
- ユーザー登録・ログイン機能
- 料理記録のCRUD（料理画像/所要時間/日付/参考レシピのURL/メモ）
- 料理記録の一覧表示
- 料理記録のアクティビティカレンダー（月/年）

### 今後追加を検討している機能
- 料理画像データから料理をした日付を自動入力
- X, Instagramのハッシュタグ検索が出来るボタンの設置（#簡単レシピ/#節約ごはん など）
- LINE通知機能
- 買い物リスト機能（食材の最安値のメモなどもできると尚可）
- 料理記録のシェア機能
- サマリレポート機能
