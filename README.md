## サービス名
CookLog（クックログ）

## サービス概要
- 「平日夜でも自炊したい」と思っているのになかなか続かない社会人向けの料理習慣化サポートアプリ
- 頑張って自炊を継続することにフォーカスします
- 料理記録を残しやすくし、カレンダーや検索で振り返りやすくすることで、自分の自炊の「続いている感」「できている感」を見える化します。

## ユーザー層について
### 想定ユーザー
- 20〜30代の社会人
- 自炊したい気持ちはあるものの、仕事の疲れ、帰宅途中にある数々の誘惑（コンビニや外食）に打ち勝てずに自炊が出来ていない人
- 時々自炊はするものの、モチベーション不安定でなかなか続かない人

### なぜこの層を対象にしたか
- 自分自身がまさに自炊を続けたいが続かなかった人であるため
- この層は「完全に料理を放棄しているわけではない」ので、ちょっとした後押しや継続している実感があれば習慣化の余地があると想定できるため

## ユーザーの獲得について
- Xの活用、料理をする方に届くような投稿をする
	- 自らアプリのユーザーとして日々記録をしシェアする
- 友人や家族に共有する

## サービスの差別化ポイント・推しポイント
- 自炊を続けたいけど続かない、という人の習慣化をサポートするという点に特化している
- 料理ログとカレンダー表示によって自炊した日・作った料理がひと目で分かり、「ちゃんと自炊が継続できている自分」を実感できる点が大きな特徴

## 機能候補
### MVPリリース時に実装したい機能
- ユーザー登録・ログイン機能
- 料理記録のCRUD（料理画像/所要時間/日付/参考レシピのURL/メモ）
- 料理記録の一覧表示
- 料理記録のアクティビティカレンダー（月/年）

### 今後追加を検討している機能
- 料理画像データから料理をした日付を自動入力
- X, Instagramのハッシュタグ検索が出来るボタンの設置（#簡単レシピ/#節約ごはん など）
- LINE通知機能
- 買い物リスト機能（食材の最安値のメモなどもできると尚可）
- 料理記録のシェア機能
- サマリレポート機能
